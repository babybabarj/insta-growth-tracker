# How to install Insta Growth Tracker on iPhone

1. Open the deployed app link in Safari.
2. Tap the Share button.
3. Tap Add to Home Screen.
4. Rename if needed: Insta Growth Tracker.
5. Tap Add.
6. Open it from the Home Screen like a normal app.

Use Safari for Add to Home Screen. Other browsers on iPhone may open the app, but Safari is the reliable install path.

Data is stored locally in the browser/app storage on that device. If browser/app data is cleared, local tracker data can be lost unless you have a backup.

Use More -> Backup -> Export JSON Backup regularly. Save the exported file to iCloud Drive or Files.
